# ExcelFlow - Исправленная версия

## 🔧 Критические исправления

### Проблема
В последнем коммите точность работы упала с **99.2%** до **3.5%** из-за:
- Упрощения логики извлечения дат (1 паттерн вместо нескольких)
- Отсутствия нормализации номенклатуры
- Проблем с индексацией данных

### Решение
✅ **Расширен модуль date_utils.py**
- Добавлено 7 паттернов распознавания дат (вместо 1)
- Добавлена валидация года (2000-2030)
- Улучшено логирование для отладки

✅ **Улучшен formula_engine.py**
- Добавлена нормализация номенклатуры
- Создан индекс для быстрого поиска
- Добавлена проверка положительных количеств

✅ **Улучшен output_writer.py**
- Автоматическая настройка ширины столбцов
- Форматирование заголовков (жирный шрифт, выравнивание)

## 💡 Использование

### Базовый режим
```bash
python main.py --input "Материалы/файл.xlsx" --output "результат.xlsx"
```

### С сравнением с эталоном
```bash
python main.py --input "Материалы/файл.xlsx" --output "результат.xlsx" --compare
```

### С DEBUG-логированием
```bash
python main.py --input "Материалы/файл.xlsx" --output "результат.xlsx" --log-level DEBUG
```

### Полный пример
```bash
python main.py \
  --input "Материалы/СК_ТПХпродажи_1_пг_2025.xlsx" \
  --output "результат.xlsx" \
  --compare \
  --log-level DEBUG \
  --chunk-size 500
```

## 🔍 Параметры

| Параметр | Описание | Обязательный | Значение по умолчанию |
|----------|----------|--------------|----------------------|
| `--input` | Входной Excel-файл | ✅ Да | - |
| `--output` | Выходной Excel-файл | ✅ Да | - |
| `--log-level` | Уровень логирования | ❌ Нет | INFO |
| `--chunk-size` | Размер chunk для обработки | ❌ Нет | 500 |
| `--compare` | Сравнить с эталонными данными | ❌ Нет | False |

## 🐛 Отладка

### Проблема: Даты не извлекаются

```bash
# Включите DEBUG-логирование
python main.py --input "файл.xlsx" --output "результат.xlsx" --log-level DEBUG > debug.log 2>&1

# Проверьте лог
grep "Дата" debug.log
grep "PSTR" debug.log
grep "REGEX" debug.log
```

### Проблема: Нет совпадений по номенклатуре

```bash
# Проверьте индекс
python -c "
from core import load_excel_file, FormulaEngine
from config import *

df_target, df_source = load_excel_file('файл.xlsx', SHEET_NAMES['TARGET'], SHEET_NAMES['SOURCE'])
engine = FormulaEngine(df_source, SOURCE_COLUMNS)
print(f'Индекс содержит {len(engine.lookup_index)} записей')
"
```
