# ExcelCostCalculator - Финальная версия

## Описание

Автоматический расчёт стоимости закупки, прямой себестоимости и накладных расходов на основе данных из Excel.

## Возможности

- ✅ Автоматический поиск строки с заголовками
- ✅ Пропуск служебных строк
- ✅ Извлечение даты из документа
- ✅ Расчёт по формулам СУММЕСЛИМН
- ✅ Сравнение с эталонными данными
- ✅ Компактные коды ошибок (#НД, #РП)
- ✅ Детальное логирование

## Установка

```bash
pip install -r requirements.txt
```

## Использование

### Базовый режим
```bash
python main.py --input "Материалы/файл.xlsx" --output "результат.xlsx"
```

### С сравнением
```bash
python main.py --input "Материалы/файл.xlsx" --output "результат.xlsx" --compare
```

### С DEBUG-логированием
```bash
python main.py --input "Материалы/файл.xlsx" --output "результат.xlsx" --log-level DEBUG
```

## Параметры

- `--input` - входной Excel-файл (обязательный)
- `--output` - выходной Excel-файл (обязательный)
- `--log-level` - уровень логирования (DEBUG/INFO/WARNING/ERROR)
- `--chunk-size` - размер chunk для обработки (по умолчанию 500)
- `--compare` - сравнить с эталонными данными

## Легенда кодов ошибок

- `#НД` — Дата не найдена в документе
- `#РП` — Требует ручной проверки - нет совпадений в справочнике
